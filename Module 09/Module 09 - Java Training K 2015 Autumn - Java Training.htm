Лекция №11

1) Сравнение объектов. Интерфейсы Comparable, Comparator. Методы hashCode, equals, compare/compareTo, их контракты.

2) Коллекции. Иерархия. Iterable/Iterator, Collection/List/Set/Queue. Классы из ядра (ArrayList/Vector/Stack, HashSet).
